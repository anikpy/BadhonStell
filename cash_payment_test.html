💰 নগদ পরিশোধ ফিচার টেস্ট

✅ সমস্যা চিহ্নিত

সমস্যা: "নগদ পরিশোধ" ফিল্ড দেখা যাচ্ছে না

কারণ: ফিল্ডটি "💰 মোট ছাড়" সেকশনের মধ্যে আছে, যা শুধুমাত্র তখনই দেখা যাবে যখন cart এ অন্তত একটি পণ্য যুক্ত হবে।

📋 টেস্ট করার ধাপসমূহ:

  1. /admin-panel/transactions/customers/70/purchase/create/ এ যান
  2. "পণ্য সার্চ" বক্সে একটি পণ্য লিখুন (যেমন: "লোহা")
  3. সার্চ রেজাল্ট থেকে একটি পণ্য নির্বাচন করুন → "যোগ করুন" বাটনে ক্লিক করুন
  4. এখন পৃষ্ঠা নিচে স্ক্রোল করুন
  5. "💰 মোট ছাড়" সেকশন দেখা যাবে
  6. এই সেকশনের ভিতরে স্ক্রোল করুন → "💵 নগদ পরিশোধ" দেখা যাবে

🔍 ফিল্ড লোড হয়েছে কিনা যাচাই:

ব্রাউজারে ডেভেলপার টুলস (F12) খুলে Elements tab এ এগুলো খুঁজুন:

  • <div id="discount-section">
  • <h4>💵 নগদ পরিশোধ</h4>
  • <input type="text" id="cash-paid-input">

⚠️ সাধারণ ভুল:

1. পণ্য যোগ না করে সরাসরি নগদ ফিল্ড খোঁজা (পাবেন না, কারণ সেকশন hidden আছে)

2. JavaScript error থাকলে (console check করুন)

3. পৃষ্ঠা পুরোনো cache (Ctrl+F5 চাপুন)

✅ Expected Behavior:

  • Cart খালি → Discount section hidden → নগদ ফিল্ড hidden
  • Cart এ 1+ পণ্য → Discount section visible → নগদ ফিল্ড visible
  • নগদ ফিল্ডে টাকা লিখলে → real-time payment summary দেখা যাবে
